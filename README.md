# React Projeto Extensão

Este projeto foi desenvolvido utilizando **React** com **Vite** para proporcionar uma aplicação rápida, modular e escalável.  
O objetivo é criar uma interface moderna e responsiva para um e-commerce de roupas, incluindo funcionalidades como:

- Navbar com navegação entre categorias
- Componentes de login e cadastro com modal
- Carrinho de compras dinâmico
- Cards de produtos estilizados e responsivos

## 🚀 Tecnologias utilizadas
- [React](https://react.dev/) — Biblioteca para construção de interfaces
- [Vite](https://vitejs.dev/) — Ferramenta de build e desenvolvimento rápido
- [CSS3](https://developer.mozilla.org/docs/Web/CSS) — Estilização responsiva
- [JavaScript ES6+](https://developer.mozilla.org/docs/Web/JavaScript) — Lógica e interatividade
